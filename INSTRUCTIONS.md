#INSTRUCTIONS
This project is a React + TypeScript app using Vite, Tailwind CSS and data fetched from TMDB API.
Follow the steps below in order to run the app locally.

## Prerequisites

Before attempting to run this app, make sure you have
the following installed on your system:

- Node.js
- Node package manager (NPM)
- A TMDB API key, sign up on the TMDB website to get your personal key.

# Instructions on how you run the system

1. Clone the project from the GitHub repo
2. Enter "npm install" in the terminal to install the neccessary dependencies
3. Create a .env file and enter your API KEY under the line "VITE_TMDB_API_KEY=your_API_key"
4. Enter "npm run dev" in the terminal to start the app on a local server.
